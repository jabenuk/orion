◆ oriSetGLFWErrorCallback()

void oriSetGLFWErrorCallback ( oriGLFWErrorCallback  callback)

Keep in mind that this is only used when using Orionwin (GLFW abstractions). In cases where GLFW is used outside the scope of orionwin (including <glfw3.h> instead of <orionwin.h>, for instance) you'll need to manage callbacks yourself. The same applies for any other windowing API you use.

◆ oriTerminate()

void oriTerminate ( )
Warning
If you are using a third-party library to handle your OpenGL context rather than Orionwin, make sure that you call this function before the OpenGL context gets terminated. For example (if you're using GLFW), call glfwTerminate() after this function. Otherwise, the program will segfault.
