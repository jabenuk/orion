buffer should be a vertex buffer. But it does not necessarily have to be. According to the OpenGL specification:

‍[Vertex Buffer Objects] are no different from any other buffer object, and a buffer object used for Transform Feedback or asynchronous pixel transfers can be used as source values for vertex arrays.

Warning
The buffer still needs to be bound to GL_ARRAY_BUFFER when not using direct state access. So if your GL version is below 4.5 and your buffer is not bound to GL_ARRAY_BUFFER then you will get a warning in the console and the function will exit early.
